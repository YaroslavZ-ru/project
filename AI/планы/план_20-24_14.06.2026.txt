ПОЧЕМУ ЭТО НУЖНО
----------------
Декоратор timed нужен для прозрачного профилирования любого метода без
изменения его логики. safe_truncate устраняет дублирование проверок при
работе с промптами для LLM (изменение 21) и при валидации данных. Модуль
utils.py входит в финальное описание структуры проекта.

ДЕТАЛИ РЕАЛИЗАЦИИ
-----------------
Модуль НЕ импортирует ничего из src/ (нет риска циклических импортов).
Использует только stdlib: logging, functools, time.

Функция/декоратор timed:
  Сигнатура: timed(logger: logging.Logger, label: str | None = None)
  Это фабрика декораторов (параметризованный декоратор).
  При вызове обёрнутой функции:
    1. Сохранить time.monotonic() в start.
    2. Вызвать оригинальную функцию.
    3. В блоке finally (даже при исключении) вычислить elapsed и логировать
       на уровне DEBUG: "[label или func.__name__] завершён за X.XXXс"
    4. Вернуть результат оригинальной функции (или пробросить исключение).
  Сохранять __name__, __doc__, __wrapped__ через functools.wraps.

  ИЛЛЮСТРАЦИЯ (не для копирования):
    import time, functools, logging

    def timed(logger, label=None):
        def decorator(func):
            @functools.wraps(func)
            def wrapper(*args, **kwargs):
                tag = label or func.__name__
                start = time.monotonic()
                try:
                    return func(*args, **kwargs)
                finally:
                    logger.debug("[%s] завершён за %.3fс", tag, time.monotonic() - start)
            return wrapper
        return decorator

Функция safe_truncate:
  Сигнатура: safe_truncate(text: str | None, max_len: int,
                            suffix: str = "...") -> str
  Поведение:
    - Если text is None или не str: вернуть "".
    - Если len(text) <= max_len: вернуть text без изменений.
    - Иначе: вернуть text[:max(0, max_len - len(suffix))] + suffix.
    - Если max_len <= 0: вернуть "".

Функция unique_ordered:
  Сигнатура: unique_ordered(seq: list) -> list
  Удаляет дубликаты, сохраняя порядок первого вхождения.
  Если элемент unhashable (например, dict): использовать O(n^2) проверку
  через "if item not in result" вместо set, не бросать исключений.

  ИЛЛЮСТРАЦИЯ:
    def unique_ordered(seq):
        seen, result = set(), []
        for item in seq:
            try:
                if item not in seen:
                    seen.add(item)
                    result.append(item)
            except TypeError:
                if item not in result:
                    result.append(item)
        return result

    self._model_loaded = True

      logger.error("Ошибка загрузки генеративной модели: %s", exc)

  Метод _parse_response(response_text: str,
                        existing_names: set[str]) -> list[dict]:
    1. Найти в response_text последнее вхождение символа ':'.
       Если есть — взять подстроку после него.
       Если нет — использовать весь response_text.
    2. Разбить по разделителям: ',', ';', '\n' (любой из них).
    3. Для каждого кандидата:
       a. Применить strip() и убрать лишние кавычки/скобки по краям.
       b. Если длина < 2 или > 80 символов: пропустить.
       c. Проверить наличие хотя бы одного слова из cfg.generative_keywords
          в кандидате (case-insensitive, проверка подстроки):
            any(kw in candidate.lower() for kw in self._cfg.generative_keywords)
          Если нет совпадений: пропустить.
       d. slug = self._slugify(candidate)
          Если slug в existing_names: пропустить.
       e. Добавить:
          {"name": slug, "label_ru": candidate.strip(),
           "type": "string",
           "description": "Предложено генеративной моделью",
           "confidence": 0.2, "source": "generative"}
    4. Вернуть список, ограниченный self._cfg.generative_max_new_params элементами.

  Метод expand(term: str, hints: list[str],
               existing_params: list[dict],
               cfg: Config | None = None) -> list[dict]:
    Args:
      term:            исходный термин (непустая строка)
      hints:           список уточнений
      existing_params: уже найденные параметры
      cfg:             если None — использовать self._cfg
    Returns:
      Список новых параметров (может быть пустым).
    Raises:
      Ничего — все ошибки логируются, возвращается [].

    Алгоритм:
      1. Если not self._available: return []
      2. self._ensure_pipeline()
      3. Если self._pipe is None: return []
      4. existing_names = {p.get("name", "") for p in existing_params}
      5. prompt = self._build_prompt(term, hints, existing_params)
      6. Выполнить с таймаутом (cfg.generative_timeout_seconds):
           from concurrent.futures import ThreadPoolExecutor, TimeoutError
           with ThreadPoolExecutor(max_workers=1) as executor:
               future = executor.submit(self._pipe, prompt)
               try:
                   result = future.result(timeout=cfg.generative_timeout_seconds)
               except TimeoutError:
                   logger.warning("GenerativeExpander: таймаут (%ss)", cfg.generative_timeout_seconds)
                   return []
               except Exception as exc:
                   logger.error("GenerativeExpander: ошибка генерации: %s", exc)
                   return []
      7. Если result пустой или нет ["generated_text"]: logger.warning, return []
      8. generated_text = result[0]["generated_text"]
      9. new_params = self._parse_response(generated_text, existing_names)
      10. logger.info("GenerativeExpander: добавлено %d параметров", len(new_params))
      11. return new_params

Dataclass SessionEntry:
  session_id: str
  domain: str | None     — последний подтверждённый домен
  last_term: str | None  — последний термин запроса
  created_at: float      — time.monotonic() при создании
  updated_at: float      — time.monotonic() при последнем update

Класс SessionManager:
  Конструктор __init__(self, config: Config):
    self._ttl      = config.session_ttl_seconds
    self._maxsize  = config.session_cache_size
    self._interval = config.session_cleanup_interval_seconds
    self._sessions: dict[str, SessionEntry] = {}
    self._lock     = threading.Lock()
    self._last_cleanup: float = time.monotonic()

  Метод get_session(session_id: str) -> SessionEntry | None:
    Если session_id пустой или None: return None.
    С self._lock:
      Если session_id не в _sessions: return None.
      entry = _sessions[session_id].
      Если time.monotonic() - entry.updated_at > self._ttl:
        del _sessions[session_id]
        logger.debug("Сессия истекла: %s", session_id)
        return None.
      return entry.

  Метод update_session(session_id: str, domain: str | None,
                       term: str | None = None) -> None:
    Если session_id пустой или None: logger.warning, return.
    С self._lock:
      Вызвать _maybe_cleanup_unsafe().
      Если session_id не в _sessions:
        Если len(_sessions) >= self._maxsize:
          Найти запись с минимальным updated_at и удалить.
          logger.warning("SessionManager: лимит %d, удалена старейшая сессия", self._maxsize)
        _sessions[session_id] = SessionEntry(
            session_id=session_id, domain=domain, last_term=term,
            created_at=time.monotonic(), updated_at=time.monotonic()
        )
      Иначе:
        entry = _sessions[session_id]
        entry.domain     = domain
        entry.last_term  = term if term is not None else entry.last_term
        entry.updated_at = time.monotonic()
      logger.debug("Сессия обновлена: %s, domain=%s", session_id, domain)

  Метод get_domain(session_id: str) -> str | None:
    entry = self.get_session(session_id)
    return entry.domain if entry is not None else None

  Метод cleanup(self) -> int:
    С self._lock:
      return self._cleanup_unsafe()

  Метод _cleanup_unsafe(self) -> int:
    (вызывается ВНУТРИ уже захваченного self._lock)
    now = time.monotonic()
    expired = [sid for sid, e in _sessions.items()
               if now - e.updated_at > self._ttl]
    for sid in expired:
      del _sessions[sid]
    if expired:
      logger.info("SessionManager: удалено %d устаревших сессий", len(expired))
    self._last_cleanup = now
    return len(expired)

  Метод _maybe_cleanup_unsafe(self) -> None:
    (вызывается ВНУТРИ захваченного self._lock)
    Если time.monotonic() - self._last_cleanup >= self._interval:
      self._cleanup_unsafe()

  Метод session_count(self) -> int:
    return len(self._sessions)
    (нет блокировки — допустимо для чтения размера dict в CPython)

Импорты в теле модуля: logging, sqlite3, numpy as np
from pathlib import Path

Функция collect_words_from_db(db_path: Path,
                               lemmatizer) -> set[str]:
  Args:
    db_path:    путь к SQLite файлу
    lemmatizer: экземпляр Lemmatizer для лемматизации слов
  Returns:
    Множество уникальных слов (str) в нижнем регистре.

  Алгоритм:
    1. Подключиться sqlite3.connect(str(db_path)), row_factory = sqlite3.Row.
    2. SELECT term FROM concepts:
       Для каждого term — lemmatizer.lemmatize_phrase(term) -> добавить все леммы.
       Также добавить слова оригинального term.split() (без лемматизации) как запасной вариант.
    3. SELECT label_ru FROM parameters:
       Аналогично лемматизировать и добавить оригинальные слова.
    4. Закрыть соединение.
    5. Отфильтровать: оставить только непустые строки длиной >= 2.

Функция build_fallback(config: Config) -> dict:
  Args: config — загруженный Config
  Returns: {"saved_path": str, "word_count": int} при успехе
           {"error": str} при критической ошибке

  Алгоритм:
    1. Проверить наличие fasttext модели:
         model_path = Path(config.fasttext_model_path)
         Если not model_path.exists():
           logger.error("FastText модель не найдена: %s", model_path)
           return {"error": "fasttext_model_not_found"}

    2. Инициализировать Lemmatizer и собрать слова:
         lem = Lemmatizer(cache_size=config.cache_lemma_size)
         words = collect_words_from_db(Path(config.db_path), lem)
         Если words пусто: logger.warning("База пуста, fallback будет пустым")

    3. Инициализировать FastTextWrapper (БЕЗ fallback_path, чтобы не петлять):
         embedding_model = FastTextWrapper(
             model_path=model_path,
             fallback_path=None,
             cache_size=config.word_vector_cache_size,
         )

    4. Собрать словарь векторов:
         embedding_dict: dict[str, np.ndarray] = {}
         for word in sorted(words):  (sorted для детерминизма)
             vec = embedding_model.get_word_vector(word)
             if not np.all(vec == 0):
                 embedding_dict[word] = vec
         logger.info("Получено векторов: %d из %d слов", len(embedding_dict), len(words))

    5. Определить путь сохранения:
         fb_path_str = config.fallback_embeddings_path
         if fb_path_str:
             output_path = Path(fb_path_str)
         else:
             output_path = Path(config.db_path).parent / "fallback_embeddings.npy"
             logger.warning(
                 "fallback_embeddings_path не задан в конфиге. "
                 "Сохраняю в %s. После запуска обновите configs/config.json.",
                 output_path
             )

    6. Создать папку: output_path.parent.mkdir(parents=True, exist_ok=True)

    7. Сохранить:
         np.save(str(output_path), embedding_dict)
         ВАЖНО: np.save с dict сохраняет в формате object dtype (pickle внутри npy).
         FastTextWrapper._load_fallback ожидает именно этот формат:
           data = np.load(path, allow_pickle=True).item()  -> dict {str: np.ndarray}

    8. Вернуть {"saved_path": str(output_path), "word_count": len(embedding_dict)}

    При PermissionError: logger.error, return {"error": "permission_denied"}
    При любом другом Exception: logger.error, return {"error": str(exc)}

НАПОМИНАНИЕ ПОСЛЕ УСПЕШНОГО ЗАПУСКА (вывести в stdout):
  print("\n>>> СЛЕДУЮЩИЙ ШАГ: установите в configs/config.json:")
  print(f'>>>   "fallback_embeddings_path": "{output_path}"')

  Условный sys.path.insert только здесь.

КРАЙНИЕ СЛУЧАИ:
  - FastText модель не найдена: вернуть error, sys.exit(1).
  - БД недоступна: создать пустой fallback (0 слов), вывести WARNING, не падать.
  - output_path уже существует: перезаписать без предупреждения.
  - Нет прав записи: поймать PermissionError, вернуть error.
  - embedding_dict пустой: всё равно сохранить (пустой словарь),
